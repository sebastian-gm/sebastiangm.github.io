Recent Projects

S&P 500 Price Prediction Using Machine Learning

This project showcases a blend of advanced machine learning models including Lasso Regression, Support Vector Regression, and Neural Networks. I have meticulously processed and cleaned the data, conducted exploratory data analysis, and employed feature selection techniques to enhance prediction accuracy. The Lasso Regression model stands out in our findings, demonstrating exceptional predictive power with high R2 scores and low RMSE values, marking a significant stride in financial forecasting using data-driven methods.

Diverse Portfolio Optimizer: Integrating Modern Finance with Quantitative Analysis

The "Diverse Portfolio Optimizer" blends modern portfolio theory with quantitative analytics to develop a diversified investment portfolio. Utilizing Python and data analysis tools, it rigorously evaluates various asset classes, including stocks, bonds, ETFs, and commodities. This project emphasizes optimizing asset allocation for maximum returns and minimal risk, guided by in-depth trend analysis and risk-return assessments. It showcases a practical, data-driven approach to investment, balancing theoretical accuracy with real-world financial dynamics.

Dynamic Insights into London’s Bike Sharing

This project delves into London’s bike-sharing system using a Kaggle-sourced dataset. I employed Python for data preprocessing and analysis, revealing key patterns such as the impact of weather on bike usage. The findings are visualized in an interactive Tableau dashboard, focusing on user-friendly and insightful displays of complex data, including a heatmap of temperature versus wind speed. This work highlights my analytical approach in uncovering and presenting data-driven insights in urban transportation.
